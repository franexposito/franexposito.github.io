Evenge

Evenge es un sistema de gestión automática de eventos con inclusión en diferentes blog (Wordpress, Blogspot), correo electrónico y redes sociales.

Implantado sobre el PaaS Google App Engine, desarrollado en Python, usando webapp2 y Jinja.

Proyecto 1er premio en el Concurso Universitario de Software Libre (http://www.concursosoftwarelibre.org/1415/0)

Mi parte del proyecto fue realizar toda la parte backend y el panel para administrar cada entrada. Para ello utilicé PHP, Javascript y Graph API de Facebook principalmente.
